Explicit is better than implicit.
Simple is better than complex.
Complex is better than complicated.

CACHE = {}

def fibonacci(x):
    if x < 2:
        return x
    value = CACHE.get(str(x))
    if not value:
        value = fibonacci(x-1) + fibonacci(x-2)
        CACHE[str(x)] = value
    return value
